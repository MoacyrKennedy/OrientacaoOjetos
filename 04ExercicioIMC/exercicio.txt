1) Criar uma pasta 04ExercicioIMc

2) Acessar a pasta via terminal: cd 04ExercicioIMc

3) Criar um projeto via terminal: dotnet new console

4) Criar uma classe chamada Pessoa

5) Na classe pessoa adicionar os atributos: peso e altura

6) Criar um método para retornar o IMC, será um método double 
com o cálculo: peso /(altura * altura)

7) Criar um método para retornar um string depedendo do IMC:
< 18.5 - Abaixo do peso
< 25 - Peso normal
< 30 - Acima do peso
< 35 - Obesidade I
< 40 - Obesidade II
<=40 - Obesidade III

8) Criar um método de mensagem para exibir o cálculo e a situção
